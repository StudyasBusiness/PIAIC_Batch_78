## Installation Instructions

### Prerequisites

- Python 3.13 or higher
- UV package manager
- Git

### Setup Steps

1. **Clone the repository:**
   ```bash
   git clone https://github.com/StudyasBusiness/PIAIC_Batch_78.git
   cd PIAIC_Batch_78
   ```

2. **Install dependencies:**
   ```bash
   uv sync
   ```

3. **Verify Python version:**
   ```bash
   python --version
   ```

### Claude Code Setup

1. **Install Claude Code:**
   ```bash
   pip install claude-code
   ```

2. **Configure Claude Code Router:**
   - Follow the setup guide in the [Claude Code Setup Repository](https://github.com/AmmarAamir786/claude_code_setup)
   - Configure your API key and preferences

## How to Use

### For Students

1. **Explore Course Materials:**
   - Read `COURSE_PROGRESS.md` for the complete learning journey
   - Check the `Assignments/` directory for practical exercises

2. **Work on Assignments:**
   - Navigate to the `Assignments/` folder
   - Choose an assignment to work on
   - Use Claude Code for assistance and guidance

3. **Create Custom Skills:**
   - Use the skill-creation workflow documented in course materials
   - Test and validate your skills
   - Share with classmates for feedback

### For Instructors

1. **Track Student Progress:**
   - Monitor course progress in `COURSE_PROGRESS.md`
   - Review student submissions in `Assignments/`
   - Provide feedback and guidance

2. **Update Course Materials:**
   - Add new topics and resources to `COURSE_PROGRESS.md`
   - Create new assignments as needed
   - Update skill documentation

## Learning Path

The course follows a structured progression:

### Phase 1: Foundation
- Digital FTE concepts and AI transformation
- Claude Code installation and basic usage
- CLAUDE.MD file creation and importance

### Phase 2: Skills Development
- Understanding Claude Code Skills
- Downloading and using existing skills
- Creating custom skills with skill-creator-skill

### Phase 3: Practical Application
- Building CLI applications (Todo App)
- Researching real-world AI use cases
- Creating professional portfolios
- Skill validation and testing

## Assignment Overview

### Assignment 1: Python CLI Todo App
Build a command-line Todo application using Python and Claude Code assistance. Focuses on core programming concepts and CLI development.

### Assignment 2: Agentic AI Use Case Research
Research and document a real-world agentic AI implementation. Analyze problems solved, results achieved, and human-AI collaboration.

### Assignment 3: Professional Portfolio
Create a comprehensive portfolio showcasing your AI skills, projects, and learning journey.

### Assignment 4: Skill Validation
Learn to validate and test custom Claude Code skills for quality and functionality.
